# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/63336884-3d86-4c97-b265-ba124b3ffed5

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## How to run it
1. Clone this repo
2. Get a free API key from Google AI Studio (aistudio.google.com)
3. Create a .env file and add: GEMINI_API_KEY=your-key-here
4. Open index.html / run python app.py
